# flapster

Flapster is a Python game inspired by the classic Flappy Bird, developed using Pygame. In this game, the player controls a bird that continuously moves forward and must navigate through an endless series of pipes without colliding. The objective is to achieve the highest possible score by passing through as many pipes as possible. Flapster demonstrates core game development concepts such as sprite handling, collision detection, scoring, and basic animation.

## features
- Simple and addictive gameplay similar to Flappy Bird.
- Smooth animations and responsive controls.
- Score tracking and game-over detection.
- Easy to run as a standalone executable.

## installation
To play Flapster, download the repository and run the executable file `flapster.exe`. No installation or additional dependencies are required.

## how to play
- Launch `flapster.exe` to start the game.
- Use the up arrow key to make the bird jump.
- Avoid hitting the pipes and the ground.
- Try to achieve the highest score possible.
